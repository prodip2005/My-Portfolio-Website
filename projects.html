Emergency Service Calling Website

This is an interactive web application designed to simulate a realistic emergency service calling system for urgent situations. Users start with 100 virtual coins, and each emergency call deducts 20 coins from their balance. The app displays the caller’s name and phone number for each call, providing a clear and organized call log. If the user’s coin balance falls below 20, further calls are automatically disabled to ensure proper resource management. The application also includes a heart button feature that allows users to gain extra lives, enhancing interactivity and user engagement significantly. All call activities are tracked in the Caller History section, where users can see a detailed log of every call made. A Clear History button allows users to reset the log at any time, while a Copy button enables quick and convenient copying of phone numbers for ease of use. This project highlights skills in dynamic UI design, event handling, state management, and real-time updates, making it an excellent demonstration of building interactive, responsive, and user-friendly web applications.
